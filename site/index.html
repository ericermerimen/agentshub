AgentPing is a macOS menu bar app that monitors all your Claude Code sessions in real time. See status at a glance, jump to the right window instantly, get notified when agents need your input.

Download for macOS View on GitHub
$ brew install ericermerimen/tap/agentping $ sudo cp -pR $(brew --prefix)/opt/agentping/AgentPing.app /Applications/
$ curl -fsSL https://raw.githubusercontent.com/ericermerimen/agentping/main/Scripts/install-remote.sh | bash
$ tar xzf AgentPing-*.tar.gz && cp -r AgentPing.app /Applications/

Built for power users

Everything you need to manage multiple Claude Code sessions without losing your mind.

●

Live session status

See which agents are running, idle, waiting for input, or hit an error. Updated in real time via FSEvents.

↗

One-click window jump

Click any session to instantly focus its terminal window. Uses Accessibility API to find the right window even in complex setups.

⚠

Smart notifications

Get notified when an agent needs input, finishes, errors out, or burns through 80% of its context window.

▦

Project grouping

Sessions are grouped by project directory. Running 3 agents on the same repo? See them together.

⚙

Context window tracking

Visual progress bar shows how much context each session has consumed. Know when to start a fresh session.

$

Cost tracking

Optional per-session and total cost display. See your daily spend without leaving the menu bar.

⌘

Global hotkey

Press Ctrl+Option+A from anywhere to toggle the popover. No clicking required.

…

Context menu

Right-click to copy path, open transcript, pin sessions, open a terminal at the project directory, or mark sessions done.

⚡

CLI + hooks

Full CLI for scripting. Integrates with Claude Code hooks for automatic session tracking. One-click config setup.

Get started in 2 minutes

1

Install

One command. Pre-built universal binary, no Xcode needed.

brew install ericermerimen/tap/agentping
2

Add hooks

Open preferences, click "Copy Hook Config", paste into your Claude settings.

~/.claude/settings.json
3

Work

Start your Claude Code sessions as usual. AgentPing tracks them automatically.

Ctrl+Option+A

At-a-glance session status

Every session state is color-coded and prioritized so you always know what needs attention.

Needs Input Agent waiting for you
Error Something went wrong
Running Agent is working
Idle Waiting for your turn
Done Task completed
Unavailable Process not found
